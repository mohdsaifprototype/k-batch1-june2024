How do we create our script for the webpage

There are two methods to use JavaScritp in a webpage.

  1. Script element. 
    <script>we write our JavaScritp code here!<script>
    Note: we can use script element anywhere in the html element. however it's good to write it in the head element.
  2. External script. 
    <script src="filename.js">we write our JavaScritp code here!<script>
    Note: we can use script element anywhere in the html element. however it's good to write it in the head element.

JavaScritp Variables

What is a variable?

A variable is a container for storing data values.

JavaScript is a case-sensitive language.

Variable names can be short (a-z, A-Z, 0-9) or long (a-z, A-Z, 0-9, _, $, or # characters)

Note:Variable names cannot start with a number.
Variable names can only contain letters, numbers, underscores, and dollar signs.

Declaring Variables

There are two ways to declare variables in JavaScritp.

  1. Variable Declaration.
  2. Variable Initialization.

Variable Declaration:

var var_name;

Variable Initialization:

var var_name = value;
